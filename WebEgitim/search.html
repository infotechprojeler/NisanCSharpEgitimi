Arama Sonuçları

http://127.0.0.1:5500/search.html?kelime=bilgisayar bu url de search.html açılması istenen sayfa adresidir.
sonrasındaki ? işareti bu sayfaya bir query string geleceğini ifade eder
Query string ise sayfadan sayfaya adres çubuğu kullanılarak veri taşımamızı sağlayan yöntemdir
Query string 2 alandan oluşur key-value
bu örnekte (kelime=bilgisayar) kelime key, bilgisayar value dir
biz de bu key değerine ait value ye ulaşarak bu değeri veritabanı vb de aratıp ekranda çıktı oluşturabiliriz.
Query string de 1 den fazla parametre de taşıyabiliriz
Bu durumda parametrelerin arasına & işareti konulur
Örneğin : ?kelime=bilgisayar¶metre2=değer2 gibi.
