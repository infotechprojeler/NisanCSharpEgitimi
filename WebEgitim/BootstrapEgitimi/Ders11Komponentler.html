Bootstrap Icons-Web uygulamalarında küçük ikonlar kullanabilmemizi sağlayan kütüphane.

Kullanımı:

  1. Bootrstarp sitesinden icons menüsüne gidiyoruz.
  2. Aşağıda install menüsüne tıklıyoruz
  3. Uygulamamızda hızlıca kullanmak için CDN kodunu direk kopyalayabiliriz
  4. Download ile bilgisayara indirip kütüphaneyi projeye yükleyerek kullanabiliriz
  5. Css kütüphanesini ikon kullanacağımız ekranlarda sayfanın head tagları arasında çağırıyoruz
  6. Sonrasında Bootrstarp icons menüsündeki ikonlara tıklayıp açılan sayfadan i elementini classı ile birlikte kopyalıyoruz
  7. Kopyaladığımız ikonu sayfamızda kullanmak istediğimiz yere yapıştırıyoruz
