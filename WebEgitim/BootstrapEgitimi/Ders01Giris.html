Bootstrap Giriş

Bootstrap, HTML, CSS ve JavaScript kullanılarak yazılmış, açık kaynaklı ve ücretsiz bir front-end kütüphanesidir. Eski Twitter çalışanları Mark Otto ve Jacob Thornton tarafından oluşturulan bu kütüphane, geliştiricilere duyarlı web siteleri oluşturabilme imkanı sağlar.

Bootstrapi Projeye Yükleme

  1. Nuget veya Npm paket yöneticisi üzerinden
  2. Bootstrap sitesinden indirip manuel yükleyebiliriz
  3. Hazır cdn kütüphane sitelerinden alacağımız kodu sayfaya ekleyerek kullanabiliriz

Bootstrap ekranı 12 eşit parçaya bölerek kullanır. Exceldeki grid-ızgara mantığıyla elemenları konumlandırır. Css Flex alt yapısını kullanır.
