Css ile Arkaplan Resmi


Elementlere arkaplan resmi tanımlamak için style="background-image:url()" özelliği kullanılabilir.

Css ile Arkaplan Resim Tekrarı


Elementlere arkaplan resmi tanımlamak için style="background-image:url()" özelliği kullanılabilir.

background-repeat:no-repeat; ile resmin tekrarı engellenir

background-repeat:repeat-x; ile resmin x ekseninde tekrarı sağlanır

background-repeat:repeat-y; ile resmin y ekseninde tekrarı sağlanır

Css ile Arkaplan Resim Konumu


Elementlere arkaplan resmi tanımlamak için style="background-image:url()" özelliği kullanılabilir.

background-repeat:no-repeat; ile resmin tekrarı engellenir

background-repeat:repeat-x; ile resmin x ekseninde tekrarı sağlanır

background-repeat:repeat-y; ile resmin y ekseninde tekrarı sağlanır

background-position: özelliği ile arkaplan resmini çerçeve içerisinde dilediğimiz konuma ayarlayabiliriz.

Css ile Arkaplan Resim Sabitleme


Elementlere arkaplan resmi tanımlamak için style="background-image:url()" özelliği kullanılabilir.

background-repeat:no-repeat; ile resmin tekrarı engellenir

background-repeat:repeat-x; ile resmin x ekseninde tekrarı sağlanır

background-repeat:repeat-y; ile resmin y ekseninde tekrarı sağlanır

background-position: özelliği ile arkaplan resmini çerçeve içerisinde dilediğimiz konuma ayarlayabiliriz.

background-attachment: fixed; ile arkaplan resmi sabitlenir
